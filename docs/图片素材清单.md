# 图片素材清单

## 1. 文档说明

这份清单以当前仓库实际存在的素材和当前代码读取方式为准，已经同步到现在的目录结构。

- 默认同时提供 png 和 webp 两种格式，下面的路径一律省略扩展名。
- 文中“正式素材”表示当前代码已按正式资源使用。
- 文中“占位素材”表示文件存在，但在配置里仍按占位资源看待，后续可以继续替换。
- 文中“预留目录”表示目录已经建立，但当前不一定有运行时依赖。
