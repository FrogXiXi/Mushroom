# 春尽夏生 DIY 蛋糕网站图片素材清单

这份清单按后续开发可直接落地的目录结构整理。你只需要把整理好的 PNG 透明底图片放进对应文件夹即可。

## 1. 不需要准备成图片的内容

- 页面纯色背景：直接用 CSS 实现，不需要背景图。
- 标题、副标题、祝福文案：直接用字体渲染，不建议做成图片。
- 普通按钮底色、圆角、阴影：直接用 CSS 实现。
- 切割线、拖拽高亮、暗场遮罩、提示文字：直接用 Canvas 或 CSS 绘制。

## 2. 推荐图片格式

- 格式：PNG 透明底。
- 分辨率：优先 2x 图，避免模糊。
- 风格：统一为蜡笔画手绘风，避免写实照片、3D 渲染感、强商业插画感。
- 裁切：主体尽量居中，不要裁得太紧，四周留安全边距。
- 命名：统一使用英文小写加下划线，避免中文文件名。

## 3. 素材中文对照与理解说明

这一节用于把文件夹名和文件名翻译成更直观的中文，方便你整理素材时快速判断“要找什么图”。后续代码仍然按英文文件名读取，所以图片命名建议不要改中文。

## 6. 蜡笔画风素材调整规范

你现在已经明确整站都要做成蜡笔画风，所以素材方向需要从“普通手绘素材”进一步收紧成“带蜡质笔触、纸面颗粒、边缘不规整”的统一体系。下面是具体调整建议。

### 6.1 整体视觉方向

- 不要追求真实光影，改成儿童绘本感、手账感、蜡笔涂抹感。
- 所有素材边缘允许轻微不规则，不要做得像矢量图一样过于干净。
- 颜色尽量柔和偏奶油色、果酱色、粉蜡笔色，避免高饱和荧光色。
- 同一套素材里线条粗细尽量统一，建议偏粗一点，营造手绘感。

### 6.2 蛋糕素材怎么调整

- 蛋糕胚不要做得太真实，表面可以加入蜡笔横向或弧形排线。
- 奶油边缘不要太光滑，建议做轻微颗粒感和不均匀抹开痕迹。
- 阴影不要做真实投影，改成淡棕色、淡灰紫色的蜡笔扫色。
- 高光不要用纯白亮斑，改成轻浅奶白的涂抹块。

### 6.3 道具素材怎么调整

- 鸡蛋、碗、搅拌器都要改成蜡笔手绘轮廓，线条可以略抖，不要工业图标感。
- 色素瓶建议做成像小手绘滴管瓶，不要做成实验室瓶子或写实瓶子。
- 原来文档里的 brush，更适合改成“蜡笔”造型，而不是传统毛笔或水彩笔。
- 蛋糕刀建议画得更温和圆润，弱化锋利感，保持治愈氛围。

### 6.4 装饰素材怎么调整

- 水果不要写实切片，建议做成蜡笔水果贴纸感。
- 糖珠、巧克力、蝴蝶结、插牌都可以保留独立 PNG，但边缘和填色要保留笔触。
- 花朵类装饰适合做成蜡笔压色块，不要做精细花瓣纹理。
- 数字蜡烛和细长蜡烛建议加明显的手绘描边，和整站风格统一。

### 6.5 页面 UI 素材怎么调整

- 按钮图标不要用通用扁平图标，建议改成蜡笔小图标。
- 分享模板边框不要做成标准几何边框，建议做成手绘相框、手账卡片。
- 首页如果要加点缀，小爱心、小星星、小花建议全部采用蜡笔涂鸦版。
- 提示弹窗、工具栏容器、分类标签，也建议后续配合纸张纹理和蜡笔描边。

### 6.6 技术实现上要预留的配合点

- 图片本身有蜡笔颗粒后，Canvas 缩放时更容易模糊，所以尽量准备 2x 或更高分辨率素材。
- 同类装饰素材的留白边距要一致，否则拖拽摆放时视觉重心会乱。
- 所有透明底 PNG 最好都保持接近真实重心居中，方便后续旋转和切割归属判断。
- 纸张纹理、蜡笔笔触纹理适合单独做成叠加素材，不要直接烘焙进每张图里，否则后面很难统一调整。

## 7. 优先整理顺序

如果你想先把最影响开发的素材准备出来，建议按这个顺序：

1. 蛋糕胚：single、double/top、double/bottom
2. 制作流程道具：egg、food_coloring_bottles、bowl、mixer
3. 切蛋糕与仪式：cake_knife、candles/slim、candles/number
4. 装饰核心：草莓、蓝莓、青提、小雏菊、蝴蝶结、插牌
5. 截图模板：polaroid、cream_daisy、spring_summer
6. 其余装饰与辅助图标

## 8. 实际收集建议

- 同类素材尽量统一视角和笔触，不要一部分偏写实、一部分偏卡通。
- 现在既然确定是蜡笔画风，所有素材都尽量统一成“蜡笔描边 + 粗颗粒填色 + 纸面感”的表现方式。
- 装饰物尽量保持“单个元素独立透明底”，不要把多种装饰拼成一张大图。
- 如果同一种装饰准备多个样式，建议命名成 default.png、variant_01.png、variant_02.png。
