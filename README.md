# 蜡笔手绘生日纪念页

这是一个面向单个朋友使用的静态生日纪念页。网站围绕“点亮蜡烛 -> 吹灭蜡烛 -> 保存愿望 -> 点名献灯 -> 导出纪念卡”这条主线设计，不再强调分享推广。

## 当前功能

- 中文界面与蜡笔手绘风格视觉
- 输入寿星姓名、年龄、头像与主题色
- 点击点亮蜡烛，支持麦克风吹蜡烛与长按备用模式
- 本地保存愿望，愿望只存在当前浏览器
- 点名献灯，为某一支蜡烛写名字与一句心意
- 导出愿望文本
- 生成一张包含愿望与献灯信息的纪念卡图片
- 蛋糕装饰、小彩蛋、音乐与特效开关

## 运行方式

推荐用本地静态服务运行，这样麦克风与资源加载更稳定。

```bash
npm install
npm run dev
```

启动后访问：

```text
http://localhost:4173
```

如果当前机器没有 Node.js，也可以直接在 Windows PowerShell 里运行：

```powershell
.\run_local.ps1
```

如果 PowerShell 被系统策略限制，可以直接运行：

```powershell
.\run_local.cmd
```

如果你只是看静态界面，也可以直接打开 `index.html`，但麦克风识别在部分浏览器下可能受限。

## 使用流程

1. 填写名字，按需设置年龄、头像和主题色。
2. 依次点亮蜡烛。
3. 全部点亮后，对着麦克风吹气，或切换成长按吹烛模式。
4. 吹灭后，在“本地愿望”里保存愿望。
5. 在“点名献灯”里为蜡烛写上名字和一句心意。
6. 到“保存纪念卡”选择愿望并导出图片。

## 数据说明

- 愿望、本地档案、献灯记录都保存在 `localStorage`
- 不会自动上传到任何服务器
- 纪念卡图片在浏览器本地生成并下载

## 技术说明

- 纯前端实现，无后端
- 纪念卡导出使用按需加载的 `html2canvas`
- 本地静态服务使用 `serve`

- 如果要换素材，直接覆盖 `assets/images` 和 `assets/audio` 下同名文件即可。
