# Hello Kitty蜡笔风沉浸式生日互动派对网站

一个基于原生 HTML、CSS、JavaScript 开发的生日互动网页项目，围绕点亮蜡烛、许愿吹蜡烛、留言祝福、电子贺卡等流程，复刻蜡笔手绘风格的沉浸式生日仪式体验。

## 当前状态

- 已建立项目仓库与静态资源目录结构
- 视觉参考图保留在项目根目录 `refer.png`
- 详细使用说明、素材替换教程与部署步骤将在后续完善

## 开发约束

- 无需构建工具
- 无后端、无数据库、无数据上报
- 本地直接打开 `index.html` 即可运行
- 兼容 GitHub Pages 静态部署
